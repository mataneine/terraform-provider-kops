# Usage

### Build
```
go mod vendor
cp models/bindata.go vendor/k8s.io/kops/upup/models
go build -mod vendor -v -o ~/.terraform.d/plugins/terraform-provider-kops
```

### Provider
```hcl
provider "kops" {
  state_store = "s3://kops.train.state"
}
```
